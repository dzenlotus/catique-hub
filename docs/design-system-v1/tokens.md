# Design System v1 — Токены

Источник истины: `design-tokens/tokens.json` → `tools/tokens-build.ts` → `src/app/styles/tokens.generated.css`.
Типографические токены (семейства, размеры, веса) хранятся в `src/app/styles/tokens.foundation.css` вручную — без кодогенерации.

Все hex-значения измерены по референсным мокапам (image.png / image2.png / image5.png).

### 2.1 Семейства шрифтов

| CSS-переменная    | Семейство                                                                 | Применение                               |

| `--font-display`  | `"Playfair Display Variable"`, "Iowan Old Style", Georgia, serif          | Wordmark «Catique Hub» в сайдбаре       |

## 6. Анимация и переходы

Все transitions и animations ДОЛЖНЫ быть обёрнуты в `@media (prefers-reduced-motion: reduce)` (требование README § Conventions, NFR-2).

| CSS-переменная           | Значение              | Применение                              |
|--------------------------|-----------------------|-----------------------------------------|
| `--duration-instant`     | `80ms`                | Hover bg на строках, chip toggle        |
| `--duration-fast`        | `140ms`               | Появление попапов, chip expand          |
| `--duration-normal`      | `220ms`               | Открытие/закрытие модала (scale+fade)   |
| `--duration-slow`        | `350ms`               | Drag-and-drop drop animation            |
| `--easing-default`       | `cubic-bezier(0.2,0,0,1)` | Стандартный ease-out для UI        |
| `--easing-spring`        | `cubic-bezier(0.34,1.56,0.64,1)` | Лёгкая «пружина» для DnD drop |
| `--easing-linear`        | `linear`              | Индикаторы прогресса                   |

> Motion-токены хранятся в `tokens.foundation.css` вручную — они не проходят через кодогенератор (аналогично типографике).
